Getting Started

We all have worked with data at some point of our work. Now, depending on the source, sometimes the data may not be regular, smooth and uniform and requires refinement before applying for any practical use. This software lets you do that very easily and efficiently in a graphical way. Using Interactive Data Editor one can

  1. Plot any 2D and gridded 3D data.
  2. Edit/Refine data including many useful data operations.
  3. Fit a function to the data.
To get started load your data file (in plain text format and space delimitated) from menu and select the proper column. In case of 3D data the graph will be plotted as the projection for a single value of 1st axis which can be changed using the provided slider. Now, for working with multiple data simultaneously files can be added through . Otherwise, you can add other columns of the same file using the plot list dashboard. Although all the editing features are available only for one plot at a moment, others lines can be selected alternatively to edit, from the same plot utility or by pressing .

If you are on a terminal, then the software can be quickly opened to load a file running , where the for linux is and for windows its the file inside the installation folder.

After edits you can save your modified data from . When saving for first time, the application will ask for name and location of the file to save the data. Any consecutive saves will save the data in that same file. As usual use to save in a different file. If you are working with multiple files then this will save the primary working data file only. Also plots can be exported in any desired resolutions in five file formats including, pdf, jpeg, png, svg and webp.

User Interface

The main window has basically 3 parts, Menu bar, Tool bar, Plot area and the Plot list dashboard. Most of the menubar options are self-explanatory and are detailed in the following sections. Toolbar contains the column selector and slider. Appropriate columns can be set from the column selector, while setting primary editable plot and adding or removing plots is done from the Plot list dashboard. The slider, available only when working with 3D data, is used to change the X/Y value for which the corresponding 2D cut is plotted.

The Plot area is where you view the 2D plot and make your edits. It has few useful buttons which become visible when the mouse is over this area. This plot window is fully responsive, and you can select points, zoom, pan, set axis ranges, labels etc. interactively with mouse. The right click context menu, appear only when some data points are selected, can be used to quickly smooth, remove or change data.

Apart from the main window this application has 2 external windows, which are accessed from the 'windows' menu in Menubar. Using the one can view the complete 3D plot in real time as you edit the the 2D projection on main window. Two types of 3D plots are available here, surface type and wireframe type and either one can be selected using the provided menu. By default, it shows only the surface being edited, check to show all the traces. On the other hand, with one can check the data in spreadsheet like editor, with all basic excel like feature e.g. editing, sorting, applying formulas etc. This is useful if you want to edit the data directly instead of the graphical way.

A general 3D plotter for plotting surfaces is also provided and can be opened from to plot and view 3D data. It supports multiple surfaces with basic plot configurations.

Features

This software has a lot of features for various useful data operations. Check this out for a quick demo.

  1. Dragging data points. You can edit a data points by dragging it with mouse or using the keyboard Up/Down navigation keys. To drag multiple points select the points and just drag any of them. By default, data points are movable only along vertical direction. If you want to move the data points along horizontal direction too, then check .

  2. Smooth out. Smooth out data using spline and linear kernel (Moving Average) smoother. To smooth out, select the portion of the data and press or to smooth out using spline and moving average respectively. Press and hold to apply moving average multiple times. These two methods can not be applied to the end points. For that, a local quadratic regression method is added to extrapolate and smooth out by pressing . This method can be applied to intermediate data points too. These methods are also available through the right click context menu.

    In addition to manual selection, to smooth the data, one can use the Auto Smoother feature of the software to smooth out the complete data altogether. Open and chose a desired smoothing scale to get a smoothing approximation to the original data. To quickly rectify bad data points one can use the Auto Corrector option from the menu. Chose proper smoothness and cutoff for locating the bad data and the software will provide with an corrected approximation. Press or button to apply the changes

  3. Data fitting. Data can be fitted with any given functional, although at the moment this software only support fitting 2D data. Two types of data fitting options are provided with this software. First, a simple regression type fitting, to fit the data with a polynomial of any arbitrary order. Second is a general functional fitting using Levenberg-Marquardt algorithm, one can provide any desired function to fit using this method.

  4. 2D and 3D support. This software supports both 2D and 3D data editing, but for 3D editing data must be on a rectangular grid and data file must in blocks of 1st axis values. For example, check the provided sample data. Data file can have missing grid points.

  5. Edit multiple data simultaneously. You can edit multiple columns of the same or different files simultaneously. After loading a file additional files can be added through the plot list dashboard

  6. Configure plot. Configure the plot window as you like through using the plot setting from. Here you can choose the line and marker style of the plot, change legend, tittle, axis label etc. Note, this application supports LaTeX rendering, so any valid LaTeX text can be used as title, axis labels or in legends.

  7. Drag and Drop. Instead of opening through the menu, files can be opened just by simply drag and dropping inside the software window. This can be used to open a remote file by mounting the remote folder and dropping the file from there, as for simplicity no option provided to directly open remote files.

  8. Fill missing values. Go to and fill out the start, stop and step values to fill the missing values in the data. Intermediate points will be filled using a Cubic spline and outside values will just be copied from the nearest available value or by fitting a quadratic regression function. This operation can be applied to limited part of the data by on checking the option

  9. Remove bad data. Bad data points can be removed by selecting the data points and pressing or from the right-click context menu

  10. Repeat & mirror. Go to and fill the range, method and number of times to repeat/mirror and press OK. The start and end values must exist to use this feature.

  11. Filter data. You can apply basic filter/cut off to the data. and fill values and fill the column numbers separated by comma to apply the filter.

  12. Swap data. Go to to open the swapper mode. This will add a second line to swap with the primary editable one. Select the portion and press to swap values. All other features like dragging, smoothing out etc. are still usable in this mode. After you are done close this mode by . Also swapping data only between the different columns of the same file is allowed

  13. Undo/Redo. Any immediate modification can be undone by pressing or can be redone by . At this moment undo/redo only supports up to 10 steps.

  14. Change sign. Select data points and press to change sign of the selected points.

  15. Copy/Paste. You can copy all data for a particular projection and paste to others.

  16. General 3D plotter. A general plotter to plot 3D data is provided and can be accessed through

  17. Spreadsheet. This software includes a spreadsheet like editor and can be opened through with some basic features like that of MS Excel including formula support. This can be used to view, edit, copy-paste data directly.
