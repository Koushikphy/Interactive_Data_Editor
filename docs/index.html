We have all worked with data at some point of time. Depending on the source, the data may not be regular, smooth and uniform, and requires refinement before applying for any practical use. This software lets you do that very easily and efficiently in a graphical way including many others useful data operation. Interactive Data Editor let's you,

  • Plot any 2D and gridded 3D data.
  • Edit/Refine data including many useful data operations.
  • Fit a function to the data.
Take a look at the Features or Tutorial to learn about the different features of Interactive Data Editor. Download the software to get started.
Interactive Data Editor supports several useful features to work with data,

  1. Edit one or multiple points by dragging with mouse and keyboard keys.
  2. Automatically smooth data using spline and linear Kernel smoother.
  3. Fit any function to the data.
  4. Plot any 2D and gridded 3D data
  5. Fully configurable plot window with LaTeX support.
  6. Export images in five format in any resolution.
  7. Simultaneously work with multiple datasets.
  8. Fill missing values.
  9. Remove bad data points.
  10. Repeat or mirror data any required times.
  11. Filter data subject to a given condition.
  12. Swap data between two lines or surfaces.
  13. Supports undo/redo.
  14. Change sign of any selected part of the data.
  15. Spreadsheet editor with all basic excel like features including formula parsing.
