Interactive Data Editor

This Software can be used to interactively edit and plot any 2D or gridded 3D data. In case of editing a gridded 3D data, the data will be projected on a 2D-plane for any particular X or Y-value of your choice. While editing the projected 2D plot, the complete 3D plot can be viewed through the 3D viewer links. Load the data file through the button. By default edited data will be saved in the same location with a added '_new' suffix or you can save in your desired location and name through . Editing scattered 3D data is not supported as of now.

Features:
1. Dragging by mouse :

Any one or multiple points can be moved by dragging it with mouse. By default dragging by mouse will move the point only along Y axis, X coordinate will be locked. Uncheck the to move it freely along any direction. If multiple points is selected, then dragging any selected points will move all of them.

2. Edit in spreadsheet:

Along with the graphical manipulation, this software also house a powerful spreadsheet like editor with basic features similar to that of MS Excel and others. This is very useful for things like copying, inserting, sorting etc. Load the spreadsheet from the main plot window and update the data after editing.

2. Smooth out :

You can smooth out a fluctuating/spiky portion of the data using two methods here : Spline Interpolation and Moving Average method by pressing the assigned keys. Choose either one of them depending on what kind of irregularities you want to smooth out.

3. Change Sign:

This will change the Y coordinate sign of the selected points.

4. Move or Reflect :

Selected data points can be moved or reflected/mirrored in their immediate neighborhood. See the Keyboard Shortcut section for details.

5. Undo:

Any immediate changes to the plot can be undone/redone. At present only one step undo is provided.

6. Data comparison:

A second data/plot can be added for comparing the edits made to the first plot. Load the data for this through the specified button and check/uncheck the to show/hide the plot. This second plot is not editable.

KeyBoard ShortCuts:
Auto-scale both axes.
Zoom mode.
Selection mode.
Smooth Out using Spline Interpolation.
Smooth Out the selected part of the data using Moving Average method. Press & hold the button to apply the method repeatedly to make the data more smoother. Change Sign.
Undo/Redo.
Move slider up.
Move slider down.
Move selected data points to its immediate left or right.
Reflect/Mirror selected data points to its immediate left or right.
Make the data symmetric about Y value 120 in a special case.
